# Description

This is web UI layer.

## Requirements
- `nodejs`version 16.19.*
- `yarn` version 1.22.*

## Setup env variable
- copy `.env.example` file and name it `.env` or you can just run `yarn prepare` if you are in `unix` environment. That command might not work on windows.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start
```
